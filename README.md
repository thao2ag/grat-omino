## **🎲 Grat-omino *(Pickomino Digital Version)***



*Một phiên bản kỹ thuật số hiện đại của board game nổi tiếng **Pickomino**, được xây dựng trên nền tảng Python và Streamlit.*



##### **📝 Giới thiệu**

Grat-omino mô phỏng lại toàn bộ trải nghiệm đổ xúc xắc, tính toán chiến thuật và tương tác giữa các người chơi. Dự án tập trung vào trải nghiệm người dùng với giao diện tối giản, chuyên nghiệp.



##### **✨ Tính năng nổi bật**

\- **Giao diện hiện đại:** Thiết kế theo phong cách Midnight \& Steel với điểm nhấn màu đỏ đậm (Deep Red).

\- **Logic Game chuẩn xác:** Hệ thống tính điểm, chọn sâu (worms) và cơ chế cướp đĩa từ đối thủ được lập trình chặt chẽ.

\- **Hiệu ứng mượt mà:** Sử dụng CSS Custom để tạo hiệu ứng chuyển động cho xúc xắc và thông báo trạng thái.



##### **🛠 Công nghệ sử dụng**

\- **Ngôn ngữ:** Python

\- **Framework:** Streamlit

\- **Styling:** CSS3 (Custom UI/UX)



##### **🚀 Cách chạy dự án**

**1. Clone repository:**

&#x09;git clone \[https://github.com/ten-cua-ban/grat-omino.git](https://github.com/ten-cua-ban/grat-omino.git)

**2. Cài đặt thư viện:**

&#x09;pip install -r requirements.txt

**3. Khởi chạy:**

&#x09;streamlit run app.py
